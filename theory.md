##**Truy cập đường dẫn:**
|Câu lệnh |kết quả| 
|:------: |:-----:|
|   cd    | di chuyển đến thư mục home|
|   cd .. | di chuyển về thư mục trước|
|   cd -  | di chuyển về thư mục cũ   |
|cd /home | di chuyển về thư mục home |

- ls -a: hiển thị tất các các thư mục kể cả file ẩn 
- ls -l: hiển thị tất cả các qu
